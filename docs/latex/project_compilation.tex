\hypertarget{project_compilation_linux}{}\doxysubsubsection{\texorpdfstring{Linux}{Linux}}\label{project_compilation_linux}
Ensure you know which windowing system you are using ({\ttfamily X11} or {\ttfamily Wayland}). The default setup is based on {\ttfamily X11}.\hypertarget{project_compilation_switching_wayland}{}\doxysubsubsubsection{\texorpdfstring{Switching to Wayland}{Switching to Wayland}}\label{project_compilation_switching_wayland}
Modify the following Bazel {\ttfamily BUILD} files\+:


\begin{DoxyItemize}
\item {\ttfamily /lib/\+GLFW/\+BUILD}\+: Comment every instance of {\ttfamily "{}-\/\+D\+\_\+\+GLFW\+\_\+\+X11"{}} and uncomment {\ttfamily "{}-\/\+D\+\_\+\+GLFW\+\_\+\+WAYLAND"{}}.
\item {\ttfamily /lib/vulkan/\+BUILD}\+: Comment every instance of {\ttfamily "{}-\/\+D\+\_\+\+GLFW\+\_\+\+X11"{}} and uncomment {\ttfamily "{}-\/\+D\+\_\+\+GLFW\+\_\+\+WAYLAND"{}}.
