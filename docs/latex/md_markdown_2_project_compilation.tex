\hypertarget{md_markdown_2_project_compilation_autotoc_md14}{}\doxysection{\texorpdfstring{Linux}{Linux}}\label{md_markdown_2_project_compilation_autotoc_md14}
Ensure you know which windowing system you are using ({\ttfamily X11} or {\ttfamily Wayland}). The default setup is based on {\ttfamily X11}.\hypertarget{md_markdown_2_project_compilation_autotoc_md15}{}\doxysubsection{\texorpdfstring{Switching to Wayland}{Switching to Wayland}}\label{md_markdown_2_project_compilation_autotoc_md15}
Modify the following Bazel {\ttfamily BUILD} files\+:


\begin{DoxyItemize}
\item {\ttfamily /lib/\+GLFW/\+BUILD}\+: Comment every instance of {\ttfamily "{}-\/\+D\+\_\+\+GLFW\+\_\+\+X11"{}} and uncomment {\ttfamily "{}-\/\+D\+\_\+\+GLFW\+\_\+\+WAYLAND"{}}.
\item {\ttfamily /lib/vulkan/\+BUILD}\+: Comment every instance of {\ttfamily "{}-\/\+D\+\_\+\+GLFW\+\_\+\+X11"{}} and uncomment {\ttfamily "{}-\/\+D\+\_\+\+GLFW\+\_\+\+WAYLAND"{}}.
